# lita-wotd

Get the word of the day from [dictionary.com](http://dictionary.com/)

## Installation

Add lita-wotd to your Lita instance's Gemfile:

``` ruby
gem "lita-wotd"
```

## Configuration

None.

## Usage

```
you> lita wotd
lita> perspicuity: clearness or lucidity, as of a statement. - dictionary.com
```
